# MVP Scope

This document defines the first scope of PSN Trophy Tracker.

## Included In The MVP

- List games.
- List trophies by game.
- List missing trophies.
- Sort missing trophies by rarity.
- Use SQL Server for persistence.
- Use ASP.NET Core for the backend API.
- Use Angular for the frontend application.

## Not Included For Now

- Multi-user login.
- Ranking.
- PSNProfiles scraping.
- Deployment.
- Social system.
- Advanced layout.
