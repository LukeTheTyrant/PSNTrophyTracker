# PSN Trophy Tracker

## Short Description

PSN Trophy Tracker is a personal fullstack web application for tracking PlayStation games, earned trophies, missing trophies, and overall trophy progress.

This is an unofficial project and is not affiliated with Sony Interactive Entertainment or PlayStation.

## Planned Stack

- ASP.NET Core Web API
- C#
- Entity Framework Core
- SQL Server
- Angular
- TypeScript
- Docker
- Swagger / OpenAPI

## Project Goal

The goal of this project is to provide a simple local dashboard for PlayStation trophy data.

The application will help users view their games, inspect trophies by game, identify pending trophies, and sort missing trophies by rarity.
